# nostalgia
A computer inside an old jewelry box

## Inspiration
Throwing away your old jewelry box is one thing, throwing a computer inside of it is another. "Nostalgia" is a mini-computer powered by a Raspberry Pi Zero Board, placed inside a vintage jewelry box for aesthetic purposes. This concept was inspired by [need to find her @]'s boards on TikTok, which has really caught my attention as someone who loves to blend technology with creative artistry. Making an electronics project function is cool, making it function AND look cute is even cooler. 

As someone who has only recently started diving into the world of electronics through HackClub's Blueprint (coming from a software and mobile app dev background), this project stood out to me as it seemed manageable to do but challenging enough. 

## Features & Constraints 
- Max budget of 100$

## Components 

- Raspberry Pi Zero Board
- Raspberry Pi B/B+ 3.5 inch Touch Screen
- 5MP Camera Module For Raspberry Pi
