## Inspiration
Throwing away your old jewelry box is one thing, throwing a computer inside of it is another. "Nostalgia" is a mini-computer powered by a Raspberry Pi Zero Board, placed inside a vintage jewelry box for aesthetic purposes. This concept was inspired by @ubeboobey's boards on TikTok, project N.O.M.A.D and the growing trend of cyberdesks in general, which has really caught my attention as someone who loves to blend technology with creative artistry. Making an electronics project function is cool, making it function AND look cute is even cooler. 

As someone who has only recently started diving into the world of electronics through HackClub's Blueprint (coming from a software and mobile app dev background), this project stood out to me as it seemed manageable to do but challenging enough. 

## Constraints & Features 
- Max budget of 60$
- limited space of 10.6 x 9.2 cm
- Personal offline survival computer system from scratch
- Simple games  

## Components 
- Raspberry Pi Zero Board
- Raspberry Pi B/B+ 3.5 inch Touch Screen
- Black 3x4 Keypad
- Joystick controller

## Personal Computer Survival System Features 
- A collection of PDFs on plant types and survival in wildlife
- Donwloaded Youtube videos covering foundational mathematics, physics and chemistry
- A set of notes containing my favorite recipes
- A folder containing full guides in order to maintain my software and hardware skills
- Map of the world + Night Sky Map
- A small gallery of photos containing personal details and global photographs
